1시간 배치, 층층이 쌓인 관계.

1h batch를 한 평면 슬라이스로, 5h 또는 10h을 시간 축으로 쌓는다. 같은 엔티티는 시간을 관통하는 점선 column으로 표시되고, 각 슬라이스 안의 typed n-ary relation은 발광하는 막(膜)으로 응축된다. 이게 영주님 아키텍처의 temporal tiering과 episode segmentation의 시각적 기반이다.

각 수평 평면은 1시간 batch 안에 발생한 모든 typed n-ary relation을 담는다. 디스크 모양의 반투명 boundary가 batch 경계를 표시하고, 그 안에 그 시간대 vertex들이 떠 있다. 같은 엔티티가 여러 batch에 걸쳐 등장할 때 — 점선 column이 그 엔티티의 시간 trajectory를 보여준다.

Hyperedge의 색상은 relation type을 인코딩한다. 시간이 위로 올라갈수록(hot) 색이 따뜻해지고, 아래로 내려갈수록(cold) 차가워진다 — 이게 영주님이 설계한 temporal tiering의 시각적 매핑이다. 위쪽 layer는 hot tier(MVCC, in-memory row), 아래는 warm/cold tier(columnar compressed)에 대응한다.

5h 모드는 단기 episode 단위, 10h 모드는 더 긴 episodic memory consolidation 단위로 보면 된다. Episode Segmentation Theory가 찾는 "경계"는 인접 layer 사이의 relation type 분포 변화로 드러나고, ACT-R base-level activation은 같은 entity column이 받는 hyperedge 빈도와 recency로 결정된다.
